# Build

```bash
./autogen.sh --noconfigure
./configure
```

#### If make is GNU make (try `make --version` to find out) use:

```sh
make
sudo make install
```

#### If gmake is GNU make (try `gmake --version` to find out) use:

```sh
gmake
sudo gmake install
```

___*NOTE:___ CLI signature patch based heavily upon: [xar-sig.c](http://opensource.apple.com/source/xar/xar-36.1/xarsig/xar-sig.c)
